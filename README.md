## ‚ú® Features

- Search for any word  
- View **definitions**, **parts of speech**, **synonyms**, and **antonyms**  
- Pronunciation with audio  
- **Dark & Light Mode** toggle  
- Font style switcher  
- Loading indicator  
- Error handling for invalid words  
- Fully responsive design for desktop & mobile
